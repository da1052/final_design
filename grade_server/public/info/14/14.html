谷歌公共 DNS 服务 8.8.8.8 迎来 8.8.8.8 特别纪念日

 局长 发布于 2018年08月13日
收藏 5
评论 9

昨天，谷歌的公共 DNS 服务 8.8.8.8 迎来了一个特别的日子，2018年8月12日 00:30 UTC 是它8年8个月8天又8小时的生日。谷歌在其官方博客发文纪念了这一特殊的时刻。

谷歌的公共 DNS 服务虽然不像谷歌搜索或 Gmail 这些产品那样众所周知，但它一直在默默地为大家服务着，无论是印度火车站的旅行者还是偏远的南极岛 Bouvetøya 的研究人员，全世界有数亿人依靠着谷歌的这个免费 DNS 服务将 wikipedia.org 等域名转换为 208.80.154.224 等 IP 地址。


博客文章显示，据估计，全球大约 10％ 的互联网用户依赖 8.8.8.8，每天的查询量超过一万亿。谷歌表示，虽然他们为这些增长感到自豪，但真正重要的是对用户来说，它是否是一项有价值的服务？也就是说，谷歌公共 DNS 是否让用户更快地上网？它是否保护了他们的隐私？它是否有助于他们更可靠、更安全地访问互联网网站？

换句话说，8.8.8.8 是否使 DNS 和整个互联网更好？谷歌认为，答案是肯定的。

从一开始，谷歌公共 DNS 服务的一个关键目标就是让互联网更快。当他们在2007年开始这个项目时，谷歌已经让网络搜索变得更快，不过要提供更快的公共 DNS 服务仍需要一段时间。当时，大多数 DNS 查找都使用了 ISP 的解析器，而对于小缓存，他们通常必须先进行多次 DNS 查询才能返回地址。

谷歌公共 DNS 解析器的 DNS 缓存在全球范围内，且拥有数百亿条目。由于每天有数亿客户端使用它们，因此通常会返回域查询的地址而无需额外查找，从而更快地将你连接到互联网。


谷歌公共 DNS 服务解析过程示例

谷歌还在文章中介绍了它在公共 DNS 服务其他方面所做的工作，包括如何保护用户隐私、如何保护域名系统等，详情请查看原文。
