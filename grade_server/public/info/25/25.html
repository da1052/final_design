Edge 比 Chrome 快 29%？外媒：别拿过时的标准说事

 王练 发布于 2018年07月01日
收藏 3
评论 19

微软的 Edge 浏览器美国官网首页公布了一组跑分数据，显示 Edge 比 Chrome 快 29% ，比 Firefox 省电近两倍。

在速度的比较中， Edge 团队当时是通过分别在最新版本的 Microsoft Edge，Google Chrome 和 Mozilla Firefox 上运行 Apple Jetstream 1.1 基准测试而得出，测试是在运行 Windows 10 四月更新（17134）的 PC（4GB，i5-3475S）上执行。结果显示：Edge 比 Chrome 浏览器快 29％ ，比 Firefox 快19％ 。

续航测试是 Edge 团队通过测量在相同的 Surface Book 笔记本电脑上全屏播放 HTML5 视频时电池运行的时间而得出，测试使用的依然上 Windows 10 build 17134，并通过 Wifi 连接网络。结果显示：Edge 比 Firefox 的续航高出近两倍，比 Chrome 多两小时左右。

对于该数据，外媒 tekrevue 表示微软使用的电脑配置和基准测试工具都太过时。Jetsteam 1.1 这一测试基准是在 2015 年发布的，如果使用最新的基准测试工具会有不同的结果。

tekrevue 在更现代化、更高端的机器上进行了相关测试，并加入了另外一个浏览器 —— Opera 。他们使用的是2018年1月的 Speedometer 2.0 基准测试，结果发现，在同样运行 Windows 10 build 17134 的高端配置（64GB DDR4，i7-6950X）系统上，Edge 比 Chrome 和 Opera 慢约 35％ ，比 Firefox 慢约 22％ 。

而在中端配置（8GB DDR3，i5-4250U）系统上，Edge 的情况更为糟糕，比 Chrome 和 Opera 慢近 40％ ，同时与 Firefox 保持 22％ 左右的差距。
